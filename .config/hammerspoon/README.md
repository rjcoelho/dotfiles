# Keyboard shortcuts

## Applications
* ⌘ + ⇧ + 1 - 1Password 7
* ⌘ + ⇧ + b - Brave Browser
* ⌘ + ⇧ + v - Visual Studio Code
* ⌘ + ⇧ + m - Mail
* ⌘ + ⇧ + n - Brave Browser Nightly
* ⌘ + ⇧ + s - Slack
* ⌘ + ⇧ + t - iTerm
* ⌘ + ⇧ + v - SonicWall Mobile Connect
* ⌘ + ⇧ + y - Kitty
* ⌘ + ⇧ + z - Zoom.us
